# Analisis efisiensi penggunaan bahan bakar pada mobil menggunakan machine learning 

## Deskripsi
Project ini bertujuan untuk menganalisis data penggunaan bahan bakar mobil dan mengklasifikasikan kendaraan tersebut ke dalam kategori irit dan boros, serta membangun model machine learning untuk memprediksi efisiensi bahan bakar berdasarkan karakteristik mobil.

## Tujuan
- Memberikan rekomendasi mobil dengan efisiensi bahan bakar tinggi sebagai pilihan kendaraan yang lebih ramah lingkungan dan ekonomis berdasarkan hasil analisis data dan prediksi model.

## Dataset
Dataset yang digunakan berasal dari https://github.com/hananlu/basicPython/blob/master/Dataset/cars2.json dan https://github.com/hananlu/basicPython/blob/master/Dataset/cars.xlsx
